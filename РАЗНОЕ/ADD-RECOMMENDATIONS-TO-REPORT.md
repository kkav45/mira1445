## 🔧 Логика работы

### 1. Получение данных
```javascript
const analysis = data.meteo?.analyzed || data.segmentAnalysis?.[0]?.analyzed;
```

### 2. Генерация рекомендаций
```javascript
const recommendations = 
    WeatherModule.generateRecommendations(analysis, data.pilotData) || [];
```

### 3. Fallback (если нет рекомендаций)
```javascript
if (recommendations.length === 0) {
    const overallRisk = summary.overallRisk || 'low';
    
    if (overallRisk === 'high') {
        recommendations.push({
            type: 'critical',
            text: 'ВЫСОКИЙ РИСК...'
        });
    } else if (overallRisk === 'medium') {
        recommendations.push({
            type: 'warning',
            text: 'СРЕДНИЙ РИСК...'
        });
    } else {
        recommendations.push({
            type: 'success',
            text: 'БЛАГОПРИЯТНЫЕ УСЛОВИЯ...'
        });
    }
}
```

## 💡 Примечание

**Важно:** Для генерации рекомендаций используются:
1. `WeatherModule.generateRecommendations()` — основная функция
2. `analysis.summary.overallRisk` — fallback для общего риска
3. `data.pilotData` — данные наблюдений пилота

**Автоматическое создание:**
Если `WeatherModule` недоступен или не возвращает рекомендации, создаются стандартные рекомендации на основе `overallRisk`.
