## 🎯 КРИТЕРИИ ЕДИНОГО СТИЛЯ

Все кнопки переключения маршрутов должны иметь:

1. **Одинаковый размер:**
   - padding: 8px 16px
   - font-size: 12px
   - font-weight: 600
   - border-radius: 8px

2. **Одинаковые цвета:**
   - Обычная: белый фон (#ffffff), серая рамка (#e2e8f0), серый текст (#718096)
   - Hover: светло-серый фон (#f7fafc), тень
   - Active: градиент (rgba(102, 126, 234, 0.1)), синяя рамка (#667eea), синий текст (#667eea)

3. **Одинаковые эффекты:**
   - Hover: transform: translateY(-1px), box-shadow
   - Active: box-shadow с цветом бренда
   - Transition: all 0.2s

4. **Одинаковая структура:**
   - display: inline-flex
   - align-items: center
   - gap: 6px
   - white-space: nowrap

```javascript
setActiveRoute(routeId) {
    this.activeRoute = routeId;
    const container = document.getElementById('dashboardBody');
    if (container) {
        container.innerHTML = this.render();
        this.afterRender();
    }
}
```
