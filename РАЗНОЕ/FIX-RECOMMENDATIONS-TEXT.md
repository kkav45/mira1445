## 🔍 Анализ

### Источник проблемы:

**Файл:** `js/weather.js`  
**Функция:** `generateRecommendations(analyzedData, pilotData = null)`  
**Строки:** 1196-1209 (до исправления)

### Проблема:
Текст **"по скорректированным данным"** использовался **всегда**, независимо от того, вносил ли пилот фактические наблюдения или нет.

**Код до исправления:**
```javascript
// === ОБЩИЙ РИСК ===
if (summary.overallRisk === 'high') {
    recommendations.unshift({
        type: 'critical',
        icon: 'fa-ban',
        text: `<strong>ВЫСОКИЙ РИСК</strong> по скорректированным данным. Полёт не рекомендуется.`
    });
} else if (summary.overallRisk === 'medium') {
    recommendations.unshift({
        type: 'warning',
        icon: 'fa-exclamation-triangle',
        text: `<strong>СРЕДНИЙ РИСК</strong> по скорректированным данным. Будьте осторожны.`
    });
}
```

**Результат:**
- ✅ Пилот **НЕ** вносил данные → текст "по скорректированным данным" ❌
- ✅ Пилот вносил данные → текст "по скорректированным данным" ✅

---

## ✅ Решение

### 1. Проверка наличия коррекции от пилота

**Добавлено:**
```javascript
// Проверяем, были ли скорректированы данные пилотом
const hasPilotCorrection = pilotData && 
    (pilotData.ground?.length > 0 || pilotData.flight?.length > 0);
```

**Логика:**
- `pilotData.ground?.length > 0` — есть наблюдения "Сидя на земле"
- `pilotData.flight?.length > 0` — есть наблюдения "В полёте"
- `hasPilotCorrection = true` — пилот вносил данные
- `hasPilotCorrection = false` — пилот **НЕ** вносил данные

---

### 2. Динамический текст в зависимости от коррекции

**Код после исправления:**
```javascript
const dataCorrectionText = hasPilotCorrection 
    ? 'по скорректированным данным' 
    : 'по данным анализа';

if (summary.overallRisk === 'high') {
    recommendations.unshift({
        type: 'critical',
        icon: 'fa-ban',
        text: `<strong>ВЫСОКИЙ РИСК</strong> ${dataCorrectionText}. Полёт не рекомендуется.`
    });
} else if (summary.overallRisk === 'medium') {
    recommendations.unshift({
        type: 'warning',
        icon: 'fa-exclamation-triangle',
        text: `<strong>СРЕДНИЙ РИСК</strong> ${dataCorrectionText}. Будьте осторожны.`
    });
}
```

---

### 3. Дополнительная рекомендация о коррекции

**Добавлено:**
```javascript
// Добавляем рекомендацию о коррекции, если пилот вносил данные
if (hasPilotCorrection) {
    recommendations.push({
        type: 'info',
        icon: 'fa-user-check',
        text: `<strong>Данные скорректированы</strong> по фактическим наблюдениям пилота.`
    });
}
```

**Результат:**
- Если пилот вносил данные → появляется дополнительная рекомендация типа "info"
- Если пилот **НЕ** вносил данные → рекомендация не показывается
