## ✅ Требования api.met.no

| Параметр | Значение |
|----------|----------|
| **API Key** | Не требуется |
| **Лицензия** | Creative Commons (коммерческое использование разрешено) |
| **User-Agent** | ⚠️ **Обязательно** (иначе 403 Forbidden) |
| **Rate Limits** | Не документированы (требуется кэширование) |
| **Атрибуция** | Требуется указание «MET Norway» |

## 📊 Какие данные нужны для сравнения

### Обязательные параметры (все источники):

| Параметр | Open-Meteo | MET No | Пилот | Скоррект. |
|----------|------------|--------|-------|-----------|
| **Ветер 10м (м/с)** | ✅ | ✅ | ✅ | ✅ |
| **Температура (°C)** | ✅ | ✅ | ✅ | ✅ |
| **Влажность (%)** | ✅ | ✅ | ✅ | ✅ |
| **Видимость (км)** | ✅ | ❌ | ✅ | ✅ |
| **Осадки (мм/ч)** | ✅ | ✅ | ✅ | ✅ |
| **Давление (гПа)** | ✅ | ✅ | ❌ | ✅ |
| **Облачность (%)** | ✅ | ✅ | ❌ | ✅ |
| **Риск** | ✅ | ✅ | ✅ | ✅ |

### Дополнительные параметры (только Open-Meteo):

- Ветер по высотам (250, 350, 450, 550 м)
- Точка росы
- Индекс турбулентности
- Обледенение

**Вывод:** MET No предоставляет меньше авиационных параметров, но основные метео-параметры доступны для сравнения.
