**Версия:** 0.1.5.0

---

## 🔍 Текущая структура шага 3

### Уровень 1: Вкладки анализа (4 шт.)

```
┌─────────────────────────────────────────────────────────────────┐
│  Шаг 3: Пилот → Результаты                                      │
├─────────────────────────────────────────────────────────────────┤
│  [✅ Рекомендации] [📈 Графики] [📋 Детали] [⏰ Окна]          │
└─────────────────────────────────────────────────────────────────┘
```

**Проблема:** Нет вкладки для **сравнения источников данных** (Open-Meteo vs MET No vs Пилот)

---

## 🎯 Задача

Добавить визуализацию **4 источников данных**:
1. **Open-Meteo** — основной прогноз
2. **MET No** — альтернативный прогноз
3. **Пилот** — фактические данные
4. **Скоррект.** — прогноз с коррекцией
