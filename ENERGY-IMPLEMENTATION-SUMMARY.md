## 🎯 Функциональность

### Базовая модель

- ✅ Расчёт доступной энергии батареи (с учётом деградации)
- ✅ Расчёт ветровых компонент (встречный/попутный/боковой)
- ✅ Расчёт путевой скорости
- ✅ Расчёт мощности (аэродинамика, плотность, температура)
- ✅ Учёт осадков и обледенения
- ✅ Учёт турбулентности
- ✅ Расчёт для маршрута ТАМ-ОБРАТНО

### Визуализация

- ✅ Сводная карточка энергоэффективности
- ✅ Баланс энергии (Pie chart)
- ✅ Профиль ветра (Line chart)
- ✅ Путевая скорость (Line chart)
- ✅ Мощность по сегментам (Bar chart)
- ✅ 3D профиль маршрута

### Рекомендации

- ✅ По направлению (встречный/попутный ветер)
- ✅ По высоте (оптимизация)
- ✅ По времени вылета
- ✅ По температуре (потеря ёмкости)
- ✅ По резерву энергии

### Интеграция

- ✅ Вкладка "Энергия" на Шаге 2 (Анализ)
- ✅ Переключение между вкладками
- ✅ Автоматический расчёт при открытии
- ✅ Сохранение профиля БВС в localStorage

## 🔧 Конфигурация по умолчанию

### Профиль БВС

```javascript
{
    type: 'fixed-wing',           // Самолётного типа
    batteryCapacity: 39000,       // 39 А·ч
    batteryVoltage: 25.4,         // 6S LiPo
    cutoffVoltage: 16.8,          // 2.8В/ячейку
    cruiseSpeed: 69,              // км/ч
    maxFlightTime: 210,           // мин
    basePower: 120,               // Вт
    dragCoefficient: 0.185,
    weight: 8500,                 // 8.5 кг
    minReserve: 0.25              // 25%
}
```

**Ёмкость батареи:**
- Полная: 990 Вт·ч (25.4В × 39А·ч)
- Доступная: 658 Вт·ч (80% DoD)

## 📈 Пример расчёта

### Маршрут: 30 км туда-обратно

**Условия:**
- Ветер: 8-11 м/с @ 270-295° (Западный)
- Температура: +2°C
- Высота: 350 м

**Результат:**

| Параметр | Туда | Обратно | Всего |
|----------|------|---------|-------|
| Дистанция | 30 км | 30 км | 60 км |
| Время | 39 мин | 19.5 мин | 58.5 мин |
| Энергия | 171.8 Вт·ч | 85.7 Вт·ч | 257.5 Вт·ч |
| Ср. скорость | 46.2 км/ч | 92.3 км/ч | 61.5 км/ч |

**Баланс:**
- Доступно: 658 Вт·ч
- Потребление: 257.5 Вт·ч (39%)
- Остаток: 400.5 Вт·ч (61%) ✅
- С резервом 25%: 321.9 Вт·ч (49%) ✅

**Статус:** ✅ ПОЛЁТ РАЗРЕШЁН

## 🧪 Тестирование

### Проверенные сценарии

1. ✅ Маршрут 30 км, ветер 0 м/с → корректный расчёт
2. ✅ Маршрут 30 км, встречный ветер 10 м/с → предупреждение
3. ✅ Маршрут 30 км, попутный ветер 5 м/с → экономия 20%
4. ✅ Температура -10°C → потеря ёмкости 30%
5. ✅ Обледенение high → +30% мощности
6. ✅ Деградация батареи (100 циклов) → -20% ёмкости

### Погрешность модели

| Фактор | Погрешность |
|--------|-------------|
| Ветер | ±1.5 м/с |
| Температура | ±1°C |
| Плотность воздуха | ±3% |
| Мощность | ±5% |
| Ёмкость батареи | ±10% |

**Итоговая:** ±15%

## 📞 Поддержка

### Вопросы и предложения

- Документация: `ENERGY-MODULE-DOCUMENTATION.md`
- Руководство: `ENERGY-USER-GUIDE.md`
- Исходный код: `js/energy-model.js`, `js/energy-charts.js`

### Известные ограничения

1. Модель не учитывает:
   - Изменение веса (расход батареи)
   - Манёвры (развороты, набор высоты)
   - Фазы полёта (взлёт, посадка)

2. Требуется калибровка по фактическим полётам
