## 🎨 Содержимое вкладки "Анализ рисков"

### 1. Детализация рисков по категориям

**5 карточек:**
- 🌬️ Ветер — значение, риск (0-3), прогресс-бар
- 🌧️ Осадки — значение, риск (0-3), прогресс-бар
- 👁️ Видимость — значение, риск (0-3), прогресс-бар
- ❄️ Обледенение — значение, риск (0-3), прогресс-бар
- 🌪️ Турбулентность — значение, риск (0-3), прогресс-бар

### 2. Тренды и тенденции

**4 индикатора:**
- ⬆️⬇️ Ветер — изменение на X% за 3 часа
- ⬆️⬇️ Температура — изменение на X% за 3 часа
- ➖ Влажность — стабильно/изменение
- ⬆️⬇️ Видимость — ухудшение/улучшение

**Сводка трендов:**
- ⚠️ "Ветер усилится на 28%"
- ✅ "Ветер ослабеет на 15%"
- ℹ️ "Значительных изменений не ожидается"

**Точки перелома:**
- 14:00 — Ветер превысит порог 10 м/с
- 18:00 — Высокий риск обледенения

## 🧪 Тестирование

### Предусловия:
1. Открыть `desktop.html` или `index.html`
2. Построить маршрут
3. Нажать "Анализ"
4. Перейти на шаг 3 (Пилот)
5. Ввести данные пилота
6. Нажать "Применить коррекцию"

### Проверка:

**1. Вкладка "Анализ рисков":**
- [ ] Видна 3-я вкладка "📊 Анализ рисков"
- [ ] Клик переключает вкладку
- [ ] Активная вкладка подсвечена (градиент)

**2. Детализация рисков:**
- [ ] 5 карточек в ряд
- [ ] Цвета соответствуют статусу
- [ ] Прогресс-бары показывают риск
- [ ] Значения отображаются

**3. Тренды:**
- [ ] 4 индикатора (Ветер, Темп., Влажн., Видим.)
- [ ] Стрелки направления (⬆️ ⬇️ ➖)
- [ ] Проценты изменения
- [ ] Сводка трендов
- [ ] Точки перелома (если есть)

**4. Вкладка "Рекомендации":**
- [ ] Подсказка о переходе в "Анализ рисков"
- [ ] Клик по ссылке переключает вкладку

## 📊 Сценарии использования

### Сценарий 1: Пилот хочет понять причину риска

**Действия:**
1. Шаг 3 → Визуализация → "📊 Анализ рисков"
2. Смотрит детализацию
3. Видит: "Ветер: 2/3 ⚠️", "Обледенение: 2/3 ⚠️"

**Результат:**
- Понимает **почему** риск высокий
- Знает **что делать** (проверить ветер, батарею)

---

### Сценарий 2: Пилот планирует время полёта

**Действия:**
1. Шаг 3 → Визуализация → "📊 Анализ рисков"
2. Смотрит тренды
3. Видит: "⬆️ +28%", "⚠️ Ветер усилится на 28%"
4. Смотрит точки перелома
5. Видит: "14:00 — Ветер превысит 10 м/с"

**Результат:**
- Планирует старт **до 14:00**
- Избегает ухудшения условий

---

### Сценарий 3: Пилот сравнивает источники

**Действия:**
1. Шаг 3 → Визуализация → "📋 Таблица"
2. Сравнивает 4 источника
3. Переходит в "📊 Анализ рисков"
4. Смотрит детализацию для скорректированного

**Результат:**
- Видит разницу между прогнозами
- Понимает точность коррекции

## ✅ Преимущества новой структуры

### До:
```
Вкладка "Рекомендации":
├─ Рекомендации (5-10 блоков)
├─ Детализация рисков (5 карточек) ← Спрятана!
└─ Тренды (4 индикатора + сводка) ← Спрятана!

Проблема: Всё в одной куче, сложно найти
```

### После:
```
Вкладка "Рекомендации":
└─ Рекомендации (5-10 блоков)
   └─ Подсказка: "Перейдите в Анализ рисков"

Вкладка "📊 Анализ рисков":
├─ Детализация рисков (5 карточек)
└─ Тренды (4 индикатора + сводка + точки)

Преимущество: Чёткое разделение!
```
