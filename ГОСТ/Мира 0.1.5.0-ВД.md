# 5. ПОРЯДОК ИСПОЛЬЗОВАНИЯ КОМПЛЕКТА

## 5.1. Для различных категорий пользователей

### Для пилотов БВС (операторов):
1. Начните с **Руководства оператора (ЭО)**
2. При необходимости обратитесь к **Описанию применения (ИП)**
3. Для понимания расчётов используйте **Пояснительную записку (ПЗ)**

### Для системных администраторов:
1. Изучите **Руководство системного программиста (СП)**
2. Обратитесь к **Техническому заданию (ТЗ)** для понимания требований
3. Используйте **Пояснительную записку (ПЗ)** для понимания архитектуры

### Для разработчиков:
1. Начните с **Руководства программиста (ПИ)**
2. Изучите **Пояснительную записку (ПЗ)**
3. Обратитесь к **Техническому заданию (ТЗ)** для понимания требований

### для руководителей проекта:
1. Изучите **Техническое задание (ТЗ)**
2. Ознакомьтесь с **Описанием применения (ИП)**
3. Используйте **Ведомость документов (ВД)** для контроля полноты
