**Рекомендации по хранению данных:**
- Регулярно экспортируйте маршруты в KML
- Делайте резервные копии localStorage
- Сохраняйте важные отчёты в PDF

## 8.2. Утилизация

**Утилизация не требуется** — веб-приложение не имеет физических компонентов.

**При прекращении использования:**
1. Очистите данные браузера (localStorage)
2. Удалите закладки
3. При необходимости удалите файлы с компьютера

**Очистка данных:**
```javascript
// В консоли браузера (F12):
Object.keys(localStorage)
    .filter(key => key.startsWith('mira_'))
    .forEach(key => localStorage.removeItem(key));
```

# 9. ОСОБЫЕ ОТМЕТКИ

## 9.1. Предупреждения

**⚠️ ВАЖНО:**

1. **Только для планирования!**
   Система предназначена для предполётного планирования и не должна использоваться как единственный источник метеорологической информации во время полёта.

2. **Не является сертифицированным средством**
   Система не сертифицирована для использования в авиации общего назначения или коммерческой авиации.

3. **Проверяйте данные**
   Всегда проверяйте прогноз из независимых источников перед полётом.

4. **Ответственность на пилоте**
   Окончательное решение о полёте принимает пилот, основываясь на всех доступных данных.

## 9.2. Известные ограничения

| Ограничение | Влияние | Обход |
|-------------|---------|-------|
| Максимум 50 точек в маршруте | Нельзя построить очень детальные маршруты | Разбивайте на несколько маршрутов |
| Горизонт прогноза 7 дней | Нельзя планировать на больший срок | Используйте другие источники |
| Требуется интернет | Не работает офлайн | Кэшируйте данные заранее |
| localStorage ~5 МБ | Ограничение на хранение | Регулярно очищайте старые данные |
