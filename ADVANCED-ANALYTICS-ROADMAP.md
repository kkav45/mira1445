## 🎯 Рекомендации по расширению аналитики

### Приоритет 1: Критически важные (2-4 недели)

#### 1.1. Детализация рисков по категориям ⭐⭐⭐

**Текущее состояние:**
```
Риск: 3.2 (medium)
```

**Предложение:**
```
Риск: 3.2 (medium)
├─ Ветер: 2.5/3.0 ⚠️
├─ Осадки: 0.5/3.0 ✅
├─ Видимость: 0.0/3.0 ✅
├─ Обледенение: 0.0/3.0 ✅
└─ Турбулентность: 0.2/3.0 ✅
```

**Преимущества:**
- Пилот видит **конкретную причину** риска
- Можно принять **точечное решение** (например, "ветер превышен, но видимость OK")
- Упрощается **приоритизация** угроз

**Реализация:** 4-6 часов

---

#### 1.2. Тренды и тенденции ⭐⭐⭐

**Текущее состояние:**
```
Ветер: 8.2 м/с (текущий час)
```

**Предложение:**
```
Ветер: 8.2 м/с → 9.5 м/с → 11.2 м/с
       ⬆️ Ухудшение (+37% за 3 часа)

Температура: +5°C → +3°C → +1°C
            ⬇️ Понижение (-4°C за 3 часа)
            ⚠️ Риск обледенения через 2 часа
```

**Аналитика:**
- **Вектор изменения** (улучшение/ухудшение)
- **Скорость изменения** (%/час)
- **Прогноз тренда** (когда достигнет порога)
- **Точки перелома** (когда риск изменится)

**Преимущества:**
- Пилот видит **динамику**, а не срез
- Можно **спланировать** полёт с учётом трендов
- Раннее **предупреждение** об ухудшении

**Реализация:** 6-8 часов

---

#### 1.3. Статистика и исторические данные ⭐⭐⭐

**Текущее состояние:**
```
Прогноз на 26.02.2026
```

**Предложение:**
```
📊 Статистика для этой локации (февраль):
├─ Средняя температура: +2.5°C
├─ Средний ветер: 6.3 м/с
├─ Макс. ветер (месяц): 18.5 м/с
├─ Дней с осадками: 12/28
└─ Благоприятных дней: 8/28 (29%)

📈 Сравнение с климатической нормой:
├─ Температура: +0.8°C выше нормы
├─ Ветер: +1.2 м/с выше нормы
└─ Осадки: +15% выше нормы
```

**Источники данных:**
- Исторические данные Open-Meteo (бесплатно)
- Климатические нормы (WMO)
- Локальная статистика пользователей MIRA

**Преимущества:**
- Контекст для **оценки аномалий**
- **Долгосрочное планирование** полётов
- **Обучение** на исторических данных

**Реализация:** 8-12 часов

#### 1.4. Персонализированные пороги ⭐⭐⭐

**Текущее состояние:**
```
Пороги (единые для всех):
├─ Ветер: ≤10 м/с
├─ Видимость: ≥5 км
└─ Осадки: ≤1.4 мм/ч
```

**Предложение:**
```
Профиль пилота:
├─ Тип БВС: Квадрокоптер 2кг
├─ Опыт: 150 часов
├─ Персональные пороги:
│  ├─ Ветер: ≤8 м/с (строже)
│  ├─ Видимость: ≥3 км (мягче)
│  └─ Температура: -5°C...+35°C
└─ Рекомендации:
   ├─ Не летать при ветре >8 м/с
   ├─ Избегать полётов ночью
   └─ Проверять батарею при T < 0°C
```

**Аналитика:**
- **Тип БВС** (мультиротор/самолёт, вес, размер)
- **Опыт пилота** (часы, сертификаты)
- **История полётов** (безопасные/проблемные)
- **Персональные коэффициенты** риска

**Преимущества:**
- **Точные рекомендации** под конкретный БВС
- Учёт **опыта пилота**
- **Безопасность** через персонализацию

**Реализация:** 10-15 часов

### Приоритет 2: Важные (1-2 месяца)

#### 2.1. Микроклимат и рельеф ⭐⭐

**Проблема:**
```
Прогноз для точки: Ветер 5 м/с
Реальность в ущелье: Ветер 12 м/с ⚠️
```

**Предложение:**
```
🏔️ Учёт рельефа местности:
├─ Высота над уровнем моря: 150м
├─ Экспозиция склона: Север
├─ Угол склона: 15°
├─ Близость к воде: 2км
└─ Коррекция ветра: +40% (ущелье)
   Итого: 5 м/с × 1.4 = 7 м/с
```

**Источники данных:**
- SRTM (рельеф, 30м разрешение)
- OpenStreetMap (вода, леса, города)
- Land cover (тип поверхности)

**Аналитика:**
- **Ветровые коридоры** (усиление в ущельях)
- **Термические потоки** (склоны, асфальт)
- **Туманы** (низменности, вода)
- **Городской эффект** (тепло, турбулентность)

**Реализация:** 20-30 часов

#### 2.4. Вертикальный профиль атмосферы ⭐⭐

**Текущее состояние:**
```
Высоты: 250м, 350м, 450м, 550м
Ветер: 8.2 → 9.5 → 11.2 → 12.5 м/с
```

**Предложение:**
```
🎯 Вертикальный анализ:
├─ 0м (земля): 8.2 м/с, +5°C, 65%
├─ 100м: 9.1 м/с, +4.5°C, 62%
├─ 250м: 10.5 м/с, +3.2°C, 58% ⚠️
├─ 400м: 12.0 м/с, +2.0°C, 55% ⚠️
└─ 550м: 13.5 м/с, +0.8°C, 52% 🔴

⚠️ Зоны риска:
├─ 250-350м: Сдвиг ветра 1.5 м/с/100м
├─ 400-500м: Температура near 0°C (обледенение)
└─ 500м+: Ветер >12 м/с (превышение)

✅ Рекомендации:
└─ Оптимальная высота: 100-200м
```

**Аналитика:**
- **Сдвиг ветра** (опасно для взлёта/посадки)
- **Температурная инверсия** (туман, стабильность)
- **Точка росы по высоте** (обледенение)
- **Турбулентность** (сдвиг >2 м/с/100м)

**Реализация:** 12-18 часов
