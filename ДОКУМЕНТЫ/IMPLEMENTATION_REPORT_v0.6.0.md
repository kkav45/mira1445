## 🎯 АЛГОРИТМ СРАВНЕНИЯ БАЗ

### Логика выбора лучшей базы:

```javascript
compareBasesForRoute(routeId) {
    const analytics = this.getRouteAnalytics(routeId);
    const bases = Object.values(analytics.analyticsByBase);
    
    // Фильтруем только доступные базы
    const allowedBases = bases.filter(b => b.status === 'allowed');
    
    if (allowedBases.length === 0) {
        return { bestBase: null, message: 'Нет доступных баз' };
    }
    
    // Лучшая база: минимальная энергия, максимальный резерв
    const bestBase = allowedBases.reduce((best, current) => {
        if (current.reservePercent > best.reservePercent) return current;
        if (current.reservePercent === best.reservePercent && 
            current.totalEnergy < best.totalEnergy) return current;
        return best;
    });
    
    return {
        bestBase: bestBase,
        comparison: bases,
        message: `Лучшая база: ${bestBase.base.name}`
    };
}
```

### Критерии выбора:

1. **Максимальный резерв энергии (%)** — приоритет №1
2. **Минимальная общая энергия (Вт·ч)** — при равном резерве
3. **Статус "allowed"** — только доступные базы
