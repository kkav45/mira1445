## 🎯 КАК ЭТО РАБОТАЕТ

### Логика переключения вкладок:

```javascript
// 1. Пользователь кликает на вкладку маршрута
onclick="DashboardTabsMeteo.setActiveRoute('route-1')"

// 2. Устанавливается активный маршрут
this.activeRoute = 'route-1';

// 3. Перерисовывается содержимое вкладки
container.innerHTML = this.render();
this.afterRender();

// 4. Отображаются данные для выбранного маршрута
const route = routes.find(r => r.id === 'route-1');
const analytics = MultiRouteModule.getRouteAnalytics('route-1');
```

## 📈 ПРИМЕР ИСПОЛЬЗОВАНИЯ

### Сценарий: 2 базы, 3 маршрута

**1. Откройте дашборд**
```
Кнопка "ДАШБОРД" → Открывается дашборд
```

**2. Вкладка "Метео"**
```
Метео → 📊 Все маршруты (общие данные)
Метео → ✅ Маршрут 1 (метео для маршрута 1)
Метео → ✅ Маршрут 2 (метео для маршрута 2)
Метео → ⚠️ Маршрут 3 (метео для маршрута 3)
```

**3. Вкладка "Сегменты"**
```
Сегменты → 📊 Все маршруты (все сегменты)
Сегменты → ✅ Маршрут 1 (сегменты маршрута 1)
Сегменты → ✅ Маршрут 2 (сегменты маршрута 2)
Сегменты → ⚠️ Маршрут 3 (сегменты маршрута 3)
```

**4. Вкладка "Энергия"**
```
Энергия → 📊 Все маршруты (общая энергия)
Энергия → ✅ Маршрут 1 (энергия маршрута 1)
Энергия → ✅ Маршрут 2 (энергия маршрута 2)
Энергия → ⚠️ Маршрут 3 (энергия маршрута 3)
```

**5. Вкладка "Мульти-маршрут"**
```
Мульти-маршрут → Обзор всех (все базы и маршруты)
Мульти-маршрут → ✅ Маршрут 1 (сравнение баз для маршрута 1)
Мульти-маршрут → ✅ Маршрут 2 (сравнение баз для маршрута 2)
Мульти-маршрут → ⚠️ Маршрут 3 (сравнение баз для маршрута 3)
```
