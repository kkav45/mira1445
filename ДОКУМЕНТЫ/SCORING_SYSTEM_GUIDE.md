## 📋 НАЗНАЧЕНИЕ

**MIRA Scoring System** — это автономная метеорологическая скоринг-система для оценки рисков полётов БВС (беспилотных воздушных судов).

**Возможности:**
- ✅ Расчёт интегрального скоринга (0-10 баллов)
- ✅ Учёт типа и категории БВС
- ✅ Профили миссий (визуальная, грузовая, экстренная, агро)
- ✅ Анализ 6 факторов риска
- ✅ Поиск благоприятных окон для полёта
- ✅ Визуализация на карте (OpenLayers)
- ✅ Временные графики (Plotly.js)
- ✅ Экспорт PDF-отчётов
- ✅ Сохранение конфигураций

## 🚀 БЫСТРЫЙ СТАРТ

### Шаг 1: Открыть файл

```
1. Откройте файл scoring.html в браузере (Chrome, Firefox, Edge)
2. Дождитесь загрузки карты и инициализации системы
```

### Шаг 2: Настроить БВС

```
1. Выберите пресет (быстрая настройка):
   - 🚁 Лёгкий мультиротор (до 7 кг)
   - ✈️ Самолётный тип (до 25 кг)
   - 📦 Тяжёлый грузовик (до 150 кг)
   - ⚙️ Своя конфигурация

2. Или настройте вручную:
   - Тип БВС: мультиротор / самолёт / вертолёт
   - Категория: Micro / Light / Medium / Heavy
   - Макс. ветер (м/с)
   - Макс. осадки (мм/ч)
   - Мин. видимость (км)
   - Температурный диапазон (°C)
```

### Шаг 3: Выбрать миссию

```
1. Выберите профиль миссии:
   - 👁️ Визуальное наблюдение (приоритет: безопасность)
   - 📦 Доставка груза (приоритет: эффективность)
   - 🚨 Экстренная миссия (приоритет: выполнение)
   - 🌾 Агро мониторинг (приоритет: точность)

2. Укажите приоритет миссии
```

### Шаг 4: Получить метео

```
1. Выберите точку на карте (клик мышью)
   ИЛИ
2. Нажмите кнопку "📍 Моё местоположение"
   ИЛИ
3. Введите координаты вручную

4. Нажмите кнопку "🔍 Получить метео"
```

### Шаг 5: Анализ скоринга

```
1. Система автоматически рассчитает скоринг
2. Просмотрите интегральный скоринг (0-10)
3. Изучите разбивку по факторам риска
4. Проверьте благоприятные окна
5. Ознакомьтесь с рекомендациями
```

### Шаг 6: Экспорт отчёта

```
1. Нажмите "📄 Экспорт PDF"
2. Файл сохранится в папку "Загрузки"
3. Имя файла: MIRA_Scoring_ГГГГ-ММ-ДД.pdf
```

## 🎯 АЛГОРИТМ СКОРИНГА

### Формула расчёта

```
IntegralScore = BaseScore × TypeCoeff × CategoryCoeff × MissionCoeff
```

### Базовый скоринг (0-3 балла за фактор)

| Фактор | Вес | Критерии |
|--------|-----|----------|
| **Ветер** | 25% | >120% макс. = 3, >100% = 2.5, >80% = 1.5, >60% = 0.5 |
| **Осадки** | 20% | >2 мм/ч = 3, >0.5 = 2, >0.1 = 1 |
| **Видимость** | 15% | <50% мин. = 3, <100% = 2, <150% = 1 |
| **Температура** | 15% | Вне диапазона = 3, ±5°C от границ = 1.5 |
| **Обледенение** | 15% | High = 3, Medium = 1.5 |
| **Турбулентность** | 10% | High = 2.5, Medium = 1 |

### Коэффициенты

#### Тип БВС
- Самолётный тип: **1.0**
- Мультиротор: **1.15**
- Вертолёт: **1.08**

#### Категория
- Micro (<0.25 кг): **0.8**
- Light (<7 кг): **1.0**
- Medium (<25 кг): **1.2**
- Heavy (<150 кг): **1.4**

#### Профиль миссии
- Визуальное наблюдение: **1.0**
- Доставка груза: **1.1**
- Экстренная миссия: **1.3**
- Агро мониторинг: **0.9**

## 🕐 БЛАГОПРИЯТНЫЕ ОКНА

### Алгоритм поиска

```
1. Для каждого часа прогноза:
   - Если Score < 3 → благоприятный час

2. Группировка последовательных часов:
   - Мин. длительность: 2 часа

3. Сортировка окон:
   - По среднему скорингу (лучшие primero)
```

### Пример вывода

```
🟢 3 марта 10:00 — 14:00
   Длительность: 4 ч · Средний скоринг: 1.8

🟢 3 марта 18:00 — 21:00
   Длительность: 3 ч · Средний скоринг: 2.1
```

## ⚠️ ВОЗМОЖНЫЕ ПРОБЛЕМЫ

### ❌ "Ошибка получения метео"

**Причина:** Нет доступа к Open-Meteo API

**Решение:**
1. Проверьте подключение к интернету
2. Попробуйте через 5 минут
3. Проверьте консоль браузера (F12)

---

### ❌ "Сначала получите метео данные"

**Причина:** Нажата кнопка "Рассчитать скоринг" без метео

**Решение:**
1. Выберите точку на карте
2. Нажмите "🔍 Получить метео"
3. Дождитесь загрузки данных

---

### ❌ Карта не загружается

**Причина:** Проблема с OpenLayers CDN

**Решение:**
1. Проверьте подключение к интернету
2. Обновите страницу (F5)
3. Очистите кэш браузера

---

### ❌ PDF не экспортируется

**Причина:** Блокировка всплывающих окон

**Решение:**
1. Разрешите всплывающие окна для файла
2. Проверьте настройки безопасности браузера
3. Попробуйте другой браузер
