# 📄 ИНСТРУКЦИЯ: Работа с разделом "Отчёт" в MIRA

**Версия:** 0.1.4.5  
**Дата:** 2 марта 2026 г.

---

## 🎯 Назначение

Раздел "Отчёт" в дашборде предназначен для:
- 📊 Формирования сводного отчёта по всем данным
- 🖨️ Печати отчёта на принтере
- 📥 Скачивания отчёта в формате PDF
- 👁️ Предпросмотра перед печатью/скачиванием

---

## 📍 Как открыть

1. Нажмите кнопку **"ДАШБОРД"** в левой панели карты
2. В верхнем ряду вкладок выберите **"Отчёт"** (вторая вкладка)

---

## ⚙️ Настройка разделов

Перед формированием отчёта выберите нужные разделы:

| Раздел | Описание |
|--------|----------|
| 🌤️ Метеопрогноз | Данные Open-Meteo (ветер, видимость, температура, осадки) |
| 🗺️ Сегменты | Таблица по сегментам маршрута с оценкой рисков |
| 🚩 Сидя на земле | Наблюдения пилота перед полётом |
| ✈️ В полёте | Наблюдения пилота во время полёта |
| 🔋 Энергорасчёт | Баланс энергии для электролёта |
| 📈 Графики | Временные ряды метеопараметров |

**По умолчанию:** все разделы включены ✅

---

## 📋 Пошаговая инструкция

### Шаг 1: Подготовка данных

Перед формированием отчёта убедитесь, что у вас есть:

- ✅ **Маршрут** — постройте маршрут на карте
- ✅ **Метеоданные** — нажмите "Анализ" для получения прогноза
- ✅ **Наблюдения** (опционально) — внесите данные в "Сидя на земле"

### Шаг 2: Формирование отчёта

1. Откройте вкладку **"Отчёт"** в дашборде
2. Выберите нужные разделы (галочками)
3. Нажмите кнопку **"📋 Сформировать отчёт"**

**Результат:**
- ✅ Отчёт сформирован
- ✅ Предпросмотр обновлён
- ✅ Открыто модальное окно с предпросмотром

### Шаг 4: Печать отчёта

1. В модальном окне нажмите **"🖨️ Печать"**
2. Откроется новое окно с отформатированным отчётом
3. В окне печати нажмите:
   - **"Печать"** — для печати на принтере
   - **"Закрыть"** — для закрытия окна печати

**Формат печати:**
- 📄 Формат: A4
- 📏 Поля: 20mm
- 🎨 Цвета: сохраняются

---

### Шаг 5: Скачивание PDF

1. В модальном окне нажмите **"📥 Скачать PDF"**
2. Файл будет загружен в папку "Загрузки"
3. Имя файла: `Название_маршрута.pdf`

**Структура PDF:**
1. Заголовок с датой формирования
2. Сводка маршрута (дистанция, время, риск)
3. Таблица рисков по сегментам
4. Детальный анализ по сегментам
5. Рекомендации
6. Приложение с исходными данными

## 🔧 Возможные проблемы

### ❌ "Сначала сформируйте отчёт"

**Причина:** Нажата кнопка "Печать" или "Скачать PDF" до формирования отчёта.

**Решение:**
1. Нажмите "📋 Сформировать отчёт"
2. Дождитесь появления предпросмотра
3. Затем нажмите нужную кнопку

---

### ❌ "Модуль экспорта не загружен"

**Причина:** Модуль `PdfExportModule` не подключён.

**Решение:**
1. Проверьте подключение файла `js/pdf-export.js`
2. Обновите страницу (F5)
3. Проверьте консоль браузера на ошибки

---

### ❌ "Разрешите открытие всплывающих окон"

**Причина:** Браузер блокирует всплывающее окно печати.

**Решение:**
1. Нажмите на значок блокировки в адресной строке
2. Выберите "Всегда разрешать всплывающие окна для этого сайта"
3. Повторите попытку печати

---

### ❌ Пустой предпросмотр

**Причина:** Нет данных для отображения.

**Решение:**
1. Убедитесь, что маршрут построен
2. Получите метеоданные (кнопка "Анализ")
3. Внесите наблюдения пилота (опционально)
4. Сформируйте отчёт заново

---

## 📊 Примеры использования

### Пример 1: Быстрая печать

**Сценарий:** Нужно быстро распечатать отчёт для брифинга

1. Открыть дашборд → вкладка "Отчёт"
2. Оставить все разделы включёнными
3. Нажать "📋 Сформировать отчёт"
4. Нажать "🖨️ Печать"
5. В окне печати выбрать принтер
6. Нажать "Печать"

**Время:** ~30 секунд

---

### Пример 2: Выборочный отчёт

**Сценарий:** Нужен отчёт только с метеоданными и сегментами

1. Открыть дашборд → вкладка "Отчёт"
2. Отключить лишние разделы:
   - ❌ Сидя на земле
   - ❌ В полёте
   - ❌ Энергорасчёт
3. Нажать "📋 Сформировать отчёт"
4. Нажать "📥 Скачать PDF"

**Результат:** PDF только с нужными разделами

---

### Пример 3: Отчёт с наблюдениями пилота

**Сценарий:** Нужно включить фактические данные пилота

1. Внести данные в "Сидя на земле"
2. Открыть дашборд → вкладка "Отчёт"
3. Убедиться, что раздел "Сидя на земле" включён ✅
4. Нажать "📋 Сформировать отчёт"
5. Проверить предпросмотр (должны быть наблюдения)
6. Нажать "📥 Скачать PDF"

**Результат:** PDF с наблюдениями пилота

## 💡 Советы

### ✅ Для лучшей печати:
- Используйте формат A4
- Включите "Сохранять цвета" в настройках печати
- Проверьте предпросмотр перед печатью

### ✅ Для компактного PDF:
- Отключите ненужные разделы
- Используйте только необходимые данные
- Графики занимают много места — отключайте, если не нужны

### ✅ Для быстрого доступа:
- Добавьте страницу в закладки
- Используйте горячие клавиши:
  - **Ctrl+P** — печать (когда окно открыто)
  - **Esc** — закрыть модальное окно

3. Очистите кэш браузера
4. Проверьте версию приложения (должна быть 0.1.4.5+)
