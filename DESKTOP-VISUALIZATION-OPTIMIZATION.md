**Дата:** 26 февраля 2026 г.
**Версия:** 0.1.5.0
**Проблема:** Контент не умещается в 40% ширину правой панели

---

## 🔍 Текущая ситуация

### Параметры правой панели:
- **Шина:** 40% экрана (min-width: 450px, max-width: 700px)
- **Высота:** 100% экрана
- **Контент:**
  - 4 карточки источников (grid 4×1)
  - 4 вкладки визуализации
  - Таблица (6 строк × 5 колонок)
  - 3 графика Plotly
  - Аккордеон (4 панели)
  - Рекомендации (5 блоков)

### Проблема:
```
┌─────────────────────────────────────────────────────────────┐
│  Экран 1920×1080                                            │
│  ┌──────────────┬─────────────────────────────────────────┐ │
│  │              │  Правая панель (768px)                  │ │
│  │   Карта      │  ┌───────────────────────────────────┐  │ │
│  │   (60%)      │  │ 4 карточки источников (180px)     │  │ │
│  │   1152px     │  ├───────────────────────────────────┤  │ │
│  │              │  │ Визуализация (4 вкладки)          │  │ │
│  │              │  │ ┌─────────────────────────────┐   │  │ │
│  │              │  │ │ Таблица (500px)             │   │  │ │
│  │              │  │ │ 6 строк × 5 колонок         │   │  │ │
```

### Вариант 2: Компактный режим карточек

**Идея:** Сжимать карточки до мини-версии.

```
Обычный режим (4 карточки в ряд):
┌──────────┬──────────┬──────────┬──────────┐
│🌤️ Open- │🇳🇴 MET   │🚩 Пилот  │⚙️ Скорр. │
│  Meteo   │   No     │          │          │
│ 🟢 2.1   │ 🟢 1.8   │ 🟡 3.5   │ 🟡 3.5   │
│ Ветер    │ Ветер    │ Ветер    │ Ветер    │
│ Темп.    │ Темп.    │ Темп.    │ Темп.    │
│ Влажн.   │ Влажн.   │ Влажн.   │ Влажн.   │
└──────────┴──────────┴──────────┴──────────┘

Компактный режим (4 карточки в 2 ряда):
┌───────────────┬───────────────┐
│🌤️ Open-Meteo │🇳🇴 MET No     │
│ 🟢 2.1        │ 🟢 1.8        │
│ ⚡ 8.2м/с     │ ⚡ 7.0м/с     │
└───────────────┴───────────────┘
┌───────────────┬───────────────┐
│🚩 Пилот       │⚙️ Скоррект.   │
│ 🟡 3.5        │ 🟡 3.5        │
│ ⚡ 10.5м/с    │ ⚡ 11.0м/с    │
└───────────────┴───────────────┘

Мини-режим (только риски):
[🌤️ 🟢2.1] [🇳🇴 🟢1.8] [🚩 🟡3.5] [⚙️ 🟡3.5]
```

**Преимущества:**
- ✅ Экономия 100-150px по вертикали
- ✅ Быстрое переключение
- ✅ Не требует модальных окон

**Недостатки:**
- ❌ Меньше информации видно сразу
- ❌ Графики всё равно не помещаются

### Вариант 4: Адаптивная сетка графиков

**Идея:** Графики перестраиваются в зависимости от ширины.

```
Ширина > 900px:          Ширина 600-900px:       Ширина < 600px:
┌──────────┬──────────┐  ┌──────────┬──────────┐  ┌──────────┐
│ Ветер    │ Темп.    │  │ Ветер    │ Темп.    │  │ Ветер    │
├──────────┼──────────┤  ├──────────┴──────────┤  ├──────────┤
│ Риски (2 колонки)     │  │ Риски (1 колонка)   │  │ Темп.    │
└──────────┴──────────┘  └──────────────────────┘  ├──────────┤
                                                   │ Риски    │
                                                   └──────────┘
```

**Преимущества:**
- ✅ Графики масштабируются
- ✅ Автоматическая адаптация
- ✅ Не требует действий от пользователя

**Недостатки:**
- ❌ На малой ширине всё равно тесно
- ❌ Сложнее в реализации
