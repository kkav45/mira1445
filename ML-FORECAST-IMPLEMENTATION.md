## 🎨 Как использовать

### Через консоль:

```javascript
// 1. Добавить наблюдение
MLAccuracyModule.addObservation(
    'open-meteo',  // Источник
    8.2,           // Прогноз
    10.5,          // Факт
    55.75,         // Широта
    37.62          // Долгота
);

// 2. Получить точность
const accuracy = MLAccuracyModule.getAccuracy('open-meteo', 30);
console.log('Точность:', accuracy.accuracy * 100 + '%');

// 3. Получить лучший источник
const best = MLAccuracyModule.getBestSource(55.75, 37.62, 30);
console.log('Лучший:', best.source, best.accuracy * 100 + '%');

// 4. Получить взвешенный прогноз
const weighted = MLAccuracyModule.getWeightedForecast([
    {source: 'open-meteo', value: 8.2},
    {source: 'met-no', value: 7.5}
], 55.75, 37.62);
console.log('Взвешенный:', weighted.value);
```

## 📊 Алгоритмы

### 1. Расчёт точности

```javascript
accuracy = 1 - (avgError / 5)

// Для ветра:
// ошибка 0 м/с → 100% точность
// ошибка 5 м/с → 0% точность
```

### 2. Взвешенный прогноз

```javascript
weightedValue = Σ(value × weight) / Σweight

// Пример:
// Open-Meteo: 8.2 м/с × 0.85 = 6.97
// MET No:     7.5 м/с × 0.62 = 4.65
// ------------------------------------
// Итого: (6.97 + 4.65) / (0.85 + 0.62) = 7.9 м/с
```

### 3. Адаптивная коррекция

```javascript
bias = Σ(forecast - actual) / n
corrected = forecast - bias

// Пример:
// Систематическая ошибка: +1.2 м/с
// Прогноз: 8.2 м/с
// Скоррект: 8.2 - 1.2 = 7.0 м/с
```

## ⚠️ Ограничения

1. **Нужны фактические данные** — без факта точность не рассчитывается
2. **Накопление данных** — требуется 10+ наблюдений для статистики
3. **Локальное хранение** — данные не синхронизируются между устройствами
4. **Упрощённая модель** — только средняя ошибка, без учёта сезона

```javascript
getSeasonalAccuracy(source, season) {
    const seasonalHistory = this.history.filter(obs => {
        const month = new Date(obs.date).getMonth();
        const obsSeason = this.getSeason(month);
        return obsSeason === season;
    });
    
    return this.calculateAccuracy(seasonalHistory);
}
```
