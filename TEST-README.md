# 🧪 Тест: Сравнение метеосервисов

**Файл:** `test-weather-services.html`

## 📋 Описание

Тестовая страница для **сравнения точности и скорости** различных метеосервисов.

Все запросы выполняются **параллельно** для объективного сравнения.

---

## 🚀 Как использовать

### 1. Откройте файл
Просто откройте `test-weather-services.html` в любом современном браузере.

### 2. Выберите координаты
- **Кликните по карте** для выбора точки
- **Или нажмите кнопку** 📍 для использования вашего местоположения

### 3. Настройте сервисы
Отметьте галочками сервисы для тестирования:
- ✅ **Open-Meteo** (основной, 10K запросов/день) — данные на высотаах 80м, 100м
- ✅ **MET Norway** (резерв, ~1K запросов/час) — точнее для Европы
- ⚠️ **7Timer!** (через CORS прокси, ~100 запросов/час) — нестабилен
- 🎯 **ГИБРИДНЫЙ** (рекомендуется) — Open-Meteo + MET Norway (лучшее из обоих!)

### 4. Запустите тест
Нажмите кнопку **«Тест»** (зелёная кнопка справа вверху).

### 5. Сравните результаты
- **Карточки сервисов** — подробные данные по каждому сервису
- **Данные на высотаах** — ветер и температура на 80м и 100м (Open-Meteo, ГИБРИДНЫЙ)
- **Сравнительная таблица** — все параметры в одной таблице
- **График** — визуальное сравнение температуры, ветра и влажности

## 📊 Отображаемые параметры

| Параметр | Единицы | Описание |
|----------|---------|----------|
| 🌡️ Температура | °C | На высоте 2 метра |

| ⏱️ Время ответа | мс | Скорость получения данных |

## 🔧 Технические детали

### CORS прокси
**7Timer! API** не поддерживает CORS, поэтому используется цепочка прокси:
1. `api.allorigins.win`
2. `corsproxy.io`
3. `thingproxy.freeboard.io`

Если один прокси не работает, автоматически используется следующий.

### Нормализация данных
Данные от всех сервисов приводятся к единому формату:
- Ветер конвертируется в **м/с** (из км/ч для Open-Meteo и 7Timer)
- Давление в **гПа**
- Температура в **°C**

### Кэширование
Результаты теста **не кэшируются** намеренно, чтобы каждый запуск был объективным.

## ⚠️ Важные замечания

### MET Norway требует валидный User-Agent

**✅ Настроено для:** `kkav45@ya.ru`

```javascript
// Строка ~920 в test-weather-services.html
'User-Agent': 'MIRA-WeatherTest/0.1.5 kkav45@ya.ru'
```

### 7Timer! не работает
**Причина:** CORS прокси недоступны или блокируются браузером.

**Решение:**
- Отключите 7Timer в настройках
- Попробуйте другой браузер
- Отключите расширения блокировки рекламы

### MET Norway возвращает ошибку 403
**Причина:** Неправильный User-Agent header.

**Решение:** Убедитесь, что в коде указан корректный User-Agent:
```javascript
'User-Agent': 'MIRA-WeatherTest/1.0 test@example.com'
```

### Open-Meteo медленно отвечает
**Причина:** Перегруженность серверов или проблемы с интернетом.

**Решение:** Повторите тест через несколько минут.

## 📈 Интерпретация результатов

### Точность данных
Сравните значения температуры, ветра и влажности между сервисами.

**Нормальные расхождения:**
- Температура: ±2°C
- Ветер: ±3 м/с
- Влажность: ±10%

**Большие расхождения** могут указывать на:
- Разные модели прогноза (GFS, ECMWF, ICON)
- Разное время обновления данных
- Локальные погодные аномалии

### Скорость ответа
- **< 500 мс** — отлично
- **500-1500 мс** — нормально
- **> 1500 мс** — медленно (возможны проблемы с сетью)

## 📝 Структура файла

```

├── HTML (структура страницы)

```
