# MIRA - Тест OpenLayers Features

Тестовая страница для демонстрации всех возможностей OpenLayers v9.2.4 для проекта MIRA.

## 🚀 Запуск

Просто откройте файл `test-openlayers-features.html` в любом современном браузере.

## 🎨 Анимация погоды

### Ветер
- **100 частиц** двигаются по направлению ветра
- **Длина линии** зависит от скорости
- **Прозрачность** пульсирует со временем
- **Направление** регулируется 0-360°

### Осадки
- **200× капель** (зависит от интенсивности)
- **Скорость падения** индивидуальна
- **Перезапуск** сверху при достижении низа

## 💡 Примеры использования

### Включить анимацию ветра:
1. Нажмите кнопку **«Ветер»**
2. Отрегулируйте скорость и направление
3. Наблюдайте за частицами на карте

### Измерить расстояние:
1. Нажмите **«Расстояние»**
2. Нарисуйте линию на карте
3. Результат отобразится в метке

### Экспорт карты:
1. Настройте вид карты
2. Нажмите **«Экспорт карты (PNG)»**
3. Файл сохранится автоматически

## 🔧 Технические детали

### Используемые технологии:
- **OpenLayers v9.2.4**
- **WebGL** для производительности
- **requestAnimationFrame** для анимации
- **WMS** для радарных данных

### Структура слоёв:
```
1. Базовые (OSM/Satellite/Terrain)
2. WMS Радар
3. WebGL Точки
4. Анимация (Ветер/Осадки)
5. Векторные (Heatmap/Cluster/Риски)
6. Измерения
```

## 📝 Заметки

- WMS Радар работает только для территории США (NEXRAD)
- Анимация использует requestAnimationFrame для плавности
- При зумировании частицы пересоздаются
- Экспорт PNG сохраняет только видимые слои

## 🎯 Применение в проекте MIRA

Эти функции могут быть использованы для:
- Визуализации **ветровых потоков** по маршруту
- Отображения **зон осадков** в реальном времени
- **Измерения дистанций** между точками маршрута
- **Анимации прогноза** на 24 часа
- **Сравнения** различных метеослоёв
