## 🏗️ Архитектура решения

### Компоненты:

**1. Карточки источников (4 шт.)**
- Иконка + название
- Риск (цветной бейдж)
- 3 метрики (ветер, темп., влажн.)
- Доверие (%)
- Клик → выбор источника

**2. Вкладки визуализации (4 шт.)**
- 📋 Таблица — сравнение всех источников
- 📈 Графики — 4 линии на Plotly
- 💡 Рекомендации — анализ расхождений
- 📋 Детали — аккордеон по источникам

**3. Динамический контент**
- Загружается при переключении
- Кэшируется для производительности

### Вкладка «Таблица»:

```
┌──────────────┬────────────┬────────┬──────────────┬─────────────┐
│ Параметр     │ Open-Meteo │ MET No │ Пилот (факт) │ Скоррект.   │
├──────────────┼────────────┼────────┼──────────────┼─────────────┤
│ Ветер (м/с)  │    8.2     │   7.5  │    10.5 ⚠️   │   11.0 ⚠️   │
│ Температура  │   +5.0°C   │  +4.2°C│    +3.0°C    │   +3.0°C    │
│ Риск         │ 🟢 2.1     │🟢 1.8  │  🟡 3.5 ⚠️   │ 🟡 3.5 ⚠️   │
└──────────────┴────────────┴────────┴──────────────┴─────────────┘
```

### Вкладка «Графики»:

- **Ветер:** 4 линии (Open-Meteo 🔵, MET No 🟢, Пилот 🟠, Скоррект. 🔴)
- **Температура:** 4 линии
- **Риски:** тепловая карта (время × источник)

### Вкладка «Рекомендации»:

1. **Расхождение прогнозов** (индикатор)
2. **Сравнение точности** (какой источник ближе к факту)
3. **Рекомендация** (какой источник использовать)
4. **Предупреждения** (критические риски)
5. **Благоприятные окна** (время для полёта)

### Вкладка «Детали»:

**Аккордеон (4 панели):**
- Open-Meteo (таблица по времени)
- MET No (таблица по времени)
- Пилот (фактические параметры)
- Скоррект. (данные с коррекцией)
