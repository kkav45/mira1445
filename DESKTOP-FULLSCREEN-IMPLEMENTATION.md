# ✅ Оптимизация пространства: Расширенный анализ (Desktop)

**Дата:** 26 февраля 2026 г.
**Версия:** 0.1.5.0
**Статус:** ✅ Реализовано

---

## 📊 Проблема

**Правая панель (40% экрана) не вмещает весь контент:**
- 4 карточки источников
- 4 вкладки визуализации
- Таблица (6 строк × 5 колонок)
- 3 графика Plotly
- Аккордеон (4 панели)
- Рекомендации (5 блоков)

**При ширине 768px контент требует 900-1200px.**

---

## 🎯 Реализованные решения

### 1. Увеличение ширины панели ⭐

**Файл:** `css/desktop.css`

```css
.side-panel {
    width: 50%;        /* Было 40% */
    min-width: 600px;  /* Было 450px */
    max-width: 900px;  /* Было 700px */
}
```

**Результат:**
- +200-300px полезного пространства
- Карточки источников не сжимаются
- Таблица показывает все колонки

---

### 2. Кнопка "На весь экран" ⭐

**Файл:** `js/wizard.js`

Добавлена кнопка над вкладками визуализации:

```html
<button onclick="WizardModule.openFullscreen()">
    <i class="fas fa-expand"></i> На весь экран
</button>
```

**Функции:**

```javascript

    // Закрывает модальное окно

```

---

### 3. Модальное окно на весь экран

**Файл:** `desktop.html`

```html
<div id="fullscreenModal">
    <div style="header с кнопками">
        <button>← Назад</button>
        <h2>🔬 Расширенный анализ</h2>
        <button>✕ Закрыть</button>
    </div>
    <div id="fullscreenModalBody">
        <!-- Контент копируется динамически -->
    </div>
</div>
```

**Преимущества:**
- Используется 100% экрана
- Графики крупные (2×2 сетка)
- Таблица показывает все колонки
- Рекомендации в 2 колонки

---

### 4. CSS стили для полного экрана

**Файл:** `css/desktop.css`

```css
/* Полноэкранный режим */
#fullscreenModal .sources-cards {
    grid-template-columns: repeat(4, 1fr); /* 4 карточки в ряд */
}

#fullscreenModal .charts-grid {
    grid-template-columns: repeat(2, 1fr); /* 2 графика в ряд */
    gap: 30px;
}

#fullscreenModal .chart-container {
    height: 450px; /* Крупные графики */
}

#fullscreenModal .discrepancy-grid {
    grid-template-columns: repeat(3, 1fr); /* 3 метрики в ряд */
}
```

**Адаптивность:**

```css
@media (max-width: 1400px) {
    #fullscreenModal .sources-cards {
        grid-template-columns: repeat(2, 1fr); /* 2×2 */
    }
    
    #fullscreenModal .charts-grid {
        grid-template-columns: 1fr; /* 1 колонка */
    }
}
```

## 📊 Метрики производительности

### Обычный режим:
- **Ширина панели:** 960px (при 1920px экране)
- **Таблица:** 5 колонок × 6 строк = 30 ячеек
- **Графики:** 2 в ряд (480px каждый)

### Полный экран:
- **Ширина контента:** 1600px (максимум)
- **Таблица:** 5 колонок × 6 строк = 30 ячеек
- **Графики:** 2×2 сетка (800px каждый)
- **Прирост места:** +67% (960px → 1600px)

---

## ⚠️ Известные ограничения

1. **Графики не перерисовываются автоматически** при изменении размера окна
2. **Данные копируются** (не синхронизируются) между режимами
3. **Esc не закрывает** модальное окно (только кнопки)
