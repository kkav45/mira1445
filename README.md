# MIRA | Метео-обеспечение полётов БВС

**Версия:** 0.1.4.1

Система метеорологического обеспечения и анализа рисков для беспилотных воздушных судов.

## 🚀 Возможности

- ✅ Получение данных с Open-Meteo API
- ✅ Анализ метеоусловий по 4 высотам (250, 350, 450, 550 м)
- ✅ Построение маршрутов по нескольким точкам
- ✅ Пространственное распределение рисков вдоль маршрута
- ✅ Вертикальный профиль рисков
- ✅ Анализ: ветер, осадки, видимость, обледенение, турбулентность
- ✅ Экспорт отчётов в PDF
- ✅ Интерактивная карта OpenLayers

## 📦 Быстрый старт

### 1. Настройка Git (первый запуск)
```bash
setup-git.bat
```

### 2. Загрузка на GitHub
```bash
deploy.bat
```

### 3. Открыть проект
Просто откройте `index.html` в браузере.

## 📊 Анализ параметров

| Параметр | Порог | Статус |
|----------|-------|--------|
| Ветер 10м | ≤10 м/с | OK |
| Ветер 500м | ≤15 м/с | OK |
| Осадки | ≤1.4 мм/ч | OK |
| Видимость | ≥5 км | OK |
| Обледенение | LOW | OK |

## 🗺️ Построение маршрута

1. Нажмите кнопку 🛤️ (режим маршрута)
2. Кликните на карте — точка A (старт)
3. Добавьте ещё точки (B, C, D...)
4. Нажмите кнопку 🔍 (анализ)
5. Получите отчёт о рисках вдоль маршрута

## 📄 Экспорт отчёта

Нажмите кнопку 📄 для экспорта полного отчёта в PDF:
- Статус полёта
- Метеопараметры
- Информация о маршруте
- Графики и профили
- Рекомендации

## 🔑 API

### Open-Meteo API (основное)
- Без необходимости API ключа
- Бесплатно для некоммерческого использования
- https://open-meteo.com/

### ECMWF API (дополнительно)
- Архивы TIGGE и ERA5
- Требуется регистрация и API-ключ
- 📖 [Инструкция по настройке](ECMWF_API_SETUP.md)
