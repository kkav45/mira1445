## 📊 Что реализовано

### 1. Статистика за месяц

**Функции:**
- `getHistoricalMonth(lat, lon, year, month)` — запрос исторических данных
- `calculateMonthlyStats(historicalData)` — расчёт статистики
- `compareWithNormal(stats, lat)` — сравнение с климатической нормой

**Параметры:**
- Средняя температура (макс, мин)
- Средний ветер (макс)
- Осадки (сумма, дни с осадками)
- Благоприятные дни (>5°C, <10 м/с, без осадков)

- ☢️ УФ-индекс (1-6)

## ⚠️ Ограничения

1. **Исторические данные** — требуют запроса к API (может быть медленно)
2. **Кэш 30 дней** — данные не обновляются в реальном времени
3. **Упрощённая норма** — климатическая норма приблизительная
4. **УФ-индекс** — расчёт по сезону, не по данным
