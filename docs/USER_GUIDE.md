## 📖 О системе MIRA

**MIRA** (Метеоаналитический помощник) — система метеорологического обеспечения и анализа рисков для беспилотных воздушных судов (БВС).

### Что делает MIRA?

✅ Получает прогноз погоды от Open-Meteo API  
✅ Анализирует метеоусловия по 4 высотам (250, 350, 450, 550 м)  
✅ Строит маршруты по нескольким точкам  
✅ Рассчитывает риски вдоль маршрута (ветер, осадки, видимость, обледенение)  
✅ Находит благоприятные окна для полёта  
✅ Корректирует прогноз по фактическим данным пилота  
✅ Экспортирует отчёты в PDF

### Для кого эта система?

- **Пилоты БВС** — планирование полётов
- **Операторы дронов** — оценка метеоусловий
- **Авиамоделисты** — безопасные полёты

## 🚀 Быстрый старт

### Шаг 1: Откройте систему

**Вариант А: Онлайн-версия**
```
https://kkav45.github.io/mira0141/
```

**Вариант Б: Локально**
1. Откройте файл `index.html` в браузере
2. Разрешите доступ к геолокации (по желанию)

### Шаг 2: Получите метеоданные

1. Нажмите кнопку **«Анализ»** (или выберите точку на карте)
2. Система загрузит прогноз на 7 дней
3. Просмотрите параметры:
   - 🟢 **ПОЛЁТ РАЗРЕШЁН** — можно летать
   - 🟠 **ПОЛЁТ С ОГРАНИЧЕНИЯМИ** — будьте осторожны
   - 🔴 **ПОЛЁТ ЗАПРЕЩЁН** — не рекомендуется

### Шаг 3: Постройте маршрут (если нужно)

1. Нажмите кнопку **🛤️** (режим маршрута)
2. Кликните на карте — точка A (старт)
3. Добавьте ещё точки (B, C, D...)
4. Нажмите кнопку **🔍** (анализ маршрута)
5. Получите отчёт о рисках вдоль маршрута

### Шаг 4: Уточните прогноз (опционально)

Если у вас есть датчики или вы видите фактическую погоду:

1. Нажмите кнопку **🚩 Сидя на земле**
2. Введите данные с датчиков (или нажмите «Моё местоположение»)
3. Нажмите **«Применить коррекцию прогноза»**
4. Перейдите на вкладку **«Отчёт»** → **«С учётом корректировки»**

### Шаг 5: Экспортируйте отчёт

Нажмите кнопку **📄** для экспорта полного отчёта в PDF

3. **Ручной ввод** — введите координаты

#### 3.2 Ввод метео-данных

```
┌─────────────────────────────────────┐
│ 💨 Ветер (м/с)     [11.0]          │
│ 🧭 Направление (°) [260]           │
│ 🌡️ Температура (°C) [+3.0]         │
│ 💧 Влажность (%)   [85]            │
│ 👁️ Видимость (км)  [5.0]           │
│ ☁️ Облака (м)      [400]           │
│                                     │
│ ☑️ Туман   ☑️ Осадки                │
│                                     │
│ [💾 Применить коррекцию прогноза]  │
└─────────────────────────────────────┘
```

**Поля для ввода:**

| Параметр | Диапазон | Обязательный |
|----------|----------|--------------|
| Ветер | 0-50 м/с | Нет* |
| Направление | 0-360° | Нет |
| Температура | -40...+50°C | Нет* |
| Влажность | 0-100% | Нет* |
| Видимость | 0.1-10 км | Нет |
| Облака (база) | 0-5000 м | Нет |
| Туман | да/нет | Нет |
| Осадки | да/нет | Нет |

* — хотя бы один параметр должен быть заполнен

#### 3.3 Сравнение прогноз/факт

После применения коррекции:

```
┌─────────────────────────────────────┐
│        СРАВНЕНИЕ ПРОГНОЗА И ФАКТА   │
├─────────────────────────────────────┤
│ Параметр  │ Прогноз │ Факт  │ Δ    │
│ ──────────┼─────────┼───────┼───── │
│ Ветер     │   8.0   │ 11.0  │ +38% │
│ Температура│ +5.0°C │ +3.0°C│ -2°C │

│ Туман     │   Нет   │  Да   │ ⚠️   │
```

**Обозначения:**

- `+38%` — фактический ветер на 38% выше прогноза
- `-2°C` — температура на 2°C ниже
- `⚠️` — явление не было спрогнозировано

## 📊 Параметры и пороги

### Значения по умолчанию

| Параметр | Порог | Ед. изм. |
|----------|-------|----------|
| **Ветер (земля)** | ≤10 | м/с |
| **Ветер (высота)** | ≤15 | м/с |
| **Видимость** | ≥2 | км |
| **Осадки** | ≤1.4 | мм/ч |
| **Температура** | -10...+35 | °C |
| **Облака (база)** | ≥300 | м |
| **Влажность (обледенение)** | ≤80 | % |

### Уровни риска

| Score | Уровень | Цвет | Статус |
|-------|---------|------|--------|
| 0-1 | Low | 🟢 | Разрешён |
| 2-4 | Medium | 🟠 | С ограничениями |
| 5+ | High | 🔴 | Запрещён |

### Факторы риска

**Ветер:**
- >10 м/с → +2 балла
- >8 м/с (80% порога) → +1 балл

**Видимость:**
- <2 км → +2 балла
- <3 км (150% порога) → +1 балл

**Осадки:**
- >1.4 мм/ч → +2 балла
- >0.5 мм/ч → +1 балл

**Обледенение:**
- Высокий риск → +2 балла
- Средний риск → +1 балл

## ❓ Часто задаваемые вопросы

### Общие вопросы

**В: Сколько стоит использование MIRA?**

О: Система бесплатная. Данные предоставляются Open-Meteo API без ключа.

**В: Нужен ли интернет для работы?**

О: Да, требуется подключение к интернету для получения данных API.

**В: Можно ли использовать MIRA offline?**

О: Только для просмотра ранее загруженных данных. Новые данные требуют интернета.

**В: Как часто обновляются данные?**

О: Прогноз кэшируется на 30 минут. Для обновления нажмите «Анализ» повторно.

---

### Работа с маршрутами

**В: Как импортировать маршрут из другого приложения?**

О: Экспортируйте маршрут в формате KML и загрузите через кнопку загрузки.

**В: Сколько точек может быть в маршруте?**

О: Рекомендуется до 100 точек. Большие маршруты автоматически разбиваются на сегменты по 10 км.

**В: Как сохранить маршрут?**

О: Маршруты сохраняются автоматически при создании.

---

### Коррекция прогноза

**В: Зачем вводить фактические данные?**

О: Прогнозы Open-Meteo имеют погрешности. Фактические данные повышают точность.

**В: Как часто нужно вводить данные?**

О: Перед каждым полётом, особенно если условия изменились.

**В: Что если у меня нет датчиков?**

О: Введите визуальные наблюдения (туман, осадки) или используйте данные из других источников.

**В: На сколько вперёд применяется коррекция?**

О: 
- 0 часов: 100% коррекции
- 6 часов: 78% коррекции
- 12 часов: 61% коррекции
- 24 часа: 37% коррекции

---

### Безопасность

**В: Можно ли доверять прогнозу MIRA?**

О: MIRA — инструмент планирования. Всегда проверяйте фактические условия перед полётом.

**В: Что делать, если статус «ПОЛЁТ ЗАПРЕЩЁН»?**

О: Рекомендуется отложить полёт до улучшения условий.

**В: Как определяется риск обледенения?**

О: При температуре от +5 до -10°C и влажности >80%.

---

## ⚠️ Важные предупреждения

### Ограничения системы

1. **Только для планирования!** Не используйте для реальных полётов без проверки фактических условий

2. **Погрешность прогноза:**
   - Температура: ±1-2°C
   - Ветер: ±1-2 м/с
   - Осадки: ±20-30%
   - Видимость: ±10-15%

3. **Горизонт прогноза:** 7 дней (чем дальше, тем меньше точность)

4. **Одна точка замера:** Коррекция применяется ко всему маршруту одинаково

### Рекомендации по безопасности

✅ Всегда проверяйте фактическую погоду перед взлётом  
✅ Вводите данные с датчиков для повышения точности  
✅ Не полагайтесь на прогноз >24 часов  
✅ Проверяйте каждый сегмент маршрута отдельно  
✅ Имейте запасной аэродром/площадку
