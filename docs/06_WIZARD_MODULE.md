# WizardModule — Пошаговый мастер анализа

## 📋 Назначение

Модуль `WizardModule` управляет пошаговым процессом анализа маршрута:
- Шаг 1: Выбор даты и маршрута
- Шаг 2: Анализ метеоданных
- Шаг 3: Ввод данных пилота (коррекция)
- Шаг 4: Экспорт отчёта PDF

**Файл:** `js/wizard.js`

---

## 🪜 Структура мастера

### Шаги

| № | Название | Иконка | Описание |
|---|----------|--------|----------|
| 1 | Маршрут | fa-route | Выбор даты, времени, маршрута |
| 2 | Анализ | fa-chart-bar | Просмотр рекомендаций, графиков, деталей |
| 3 | Пилот | fa-flag | Ввод фактических данных, коррекция |
| 4 | Отчёт | fa-file-pdf | Экспорт PDF |

### Вкладки шага 2 (Анализ)

| Вкладка | Иконка | Описание |
|---------|--------|----------|
| Рекомендации | fa-clipboard-list | Список рекомендаций по сегментам |
| Графики | fa-chart-line | Визуализация данных (Plotly) |
| Детали | fa-table | Таблица почасовых данных |
| Окна | fa-clock | Благоприятные окна для полёта |

## 📊 Состояние мастера

### stepData

```javascript
stepData: {
    date: "2026-02-26",           // Дата анализа
    time: "10:00",                // Время
    route: {...},                 // Объект маршрута
    segments: [...],              // Массив сегментов
    segmentAnalysis: [...],       // Анализ по сегментам
    pilotData: {...},             // Данные пилота
    correctedAnalysis: {...}      // Скорректированный анализ
}
```

## 🔧 Основные функции

### init(options)

Инициализация мастера

```javascript
WizardModule.init({
    callbacks: {
        onStepChange: (step) => { ... }
    }
});
```

---

### goToStep(step)

Переход к указанному шагу

```javascript
WizardModule.goToStep(2);  // Переход к шагу 2
```

---

### nextStep() / prevStep()

Навигация между шагами

---

### switchAnalysisTab(tabName)

Переключение вкладок на шаге анализа

```javascript
WizardModule.switchAnalysisTab('charts');
```
