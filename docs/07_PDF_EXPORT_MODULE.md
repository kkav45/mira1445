# PdfExportModule — Экспорт отчётов в PDF

## 📋 Назначение

Модуль `PdfExportModule` генерирует PDF-отчёты о метеоусловиях и анализе рисков.

**Файл:** `js/pdf-export-2page.js`  
**Зависимость:** pdfmake v0.1.66

## 🔧 Основные функции

### generateReport(data)

Генерация PDF-отчёта

**Параметры:**
```javascript
{
    route: {...},           // Маршрут
    segments: [...],        // Сегменты
    summary: {...},         // Сводка
    recommendations: [...], // Рекомендации
    flightWindows: [...]    // Временные окна
}
```

**Использование:**
```javascript
await PdfExportModule.generateReport(reportData);
// Автоматическое скачивание файла
```
