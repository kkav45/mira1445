## 🎯 Логика работы

### 1. Получение данных источников:

```javascript
getSourcesData() {
    // Open-Meteo (оригинальный прогноз)
    const openMeteo = {
        wind: originalAnalysis.hourly[0].wind10m,
        temp: originalAnalysis.hourly[0].temp2m,
        humidity: originalAnalysis.hourly[0].humidity,
        confidence: 63
    };

    // MET No (альтернативный прогноз, пока симуляция)
    const metNo = {
        wind: openMeteo.wind * 0.85,
        temp: openMeteo.temp - 0.8,
        humidity: openMeteo.humidity + 5,
        confidence: 58
    };

    // Пилот (фактические данные)
    const pilot = {
        wind: pilotData.windSpeed,
        temp: pilotData.temp,
        humidity: pilotData.humidity,
        confidence: 100
    };

    // Скорректированный (Open-Meteo + Пилот)
    const corrected = {
        wind: correctedAnalysis.hourly[0].wind10m,
        temp: correctedAnalysis.hourly[0].temp2m,
        humidity: correctedAnalysis.hourly[0].humidity,
        confidence: 100
    };
}
```

### 2. Переключение вкладок:

```javascript
switchVizTab(vizTab) {
    this.currentVizTab = vizTab;
    
    // Обновляем стили вкладок
    document.querySelectorAll('.viz-tab').forEach(tab => {
        const isActive = tab.textContent.includes(vizTab);
        tab.style.background = isActive ? 
            'linear-gradient(135deg, #667eea 0%, #764ba2 100%)' : 'transparent';
        tab.style.color = isActive ? 'white' : '#718096';
    });
    
    // Рендерим контент
    this.renderVizContent();
}
```

### 3. Инициализация графиков:

```javascript
initVizCharts() {
    const sources = this.getSourcesData();
    const hourly = this.stepData.correctedAnalysis.hourly.slice(0, 24);
    
    // График ветра (4 линии)
    Plotly.newPlot('step3VizWindChart', [
        {x: hours, y: windData.openMeteo, name: 'Open-Meteo', line: {color: '#3b82f6'}},
        {x: hours, y: windData.metNo, name: 'MET No', line: {color: '#10b981', dash: 'dash'}},
        {x: [hours[0]], y: [sources.pilot.wind], name: 'Пилот', mode: 'markers'},
        {x: hours, y: windData.corrected, name: 'Скоррект.', line: {color: '#ef4444'}}
    ]);
}
```
