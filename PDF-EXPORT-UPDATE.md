# PDF Export 2-Page Update

## Версия: 2026022702

## Изменения

### ✅ Реализованные функции

#### 1. Реальные данные в рекомендациях
- Рекомендации генерируются через `WeatherModule.generateRecommendations()`
- Учитываются данные пилота (pilotData)
- Добавляется предупреждение о высоком риске

#### 2. Временные окна с реальными данными
- Отображаются благоприятные окна для полёта
- Показывается время начала и окончания каждого окна
- Указывается продолжительность в часах
- Подсчитывается общее количество окон

#### 3. Графики с таблицами
- **Временной ряд**: температура, ветер, осадки по времени
- **Роза ветров**: распределение по направлениям (С, СВ, В, ЮВ, Ю, ЮЗ, З, СЗ)
- **Турбулентность**: среднее и максимальное значение с уровнями

#### 4. Детализация по сегментам
- Номер сегмента
- Уровень риска (цветом)
- Координаты центра сегмента
- Ветер, температура, осадки

#### 5. Карта маршрута
- Таблица с координатами всех точек маршрута
- Дистанция, время полёта, количество сегментов
- Общий риск
- Инструкция для создания скриншота карты

### 🔧 Технические изменения

1. **`createMapSection(data)`**
   - Добавлена таблица точек маршрута
   - Добавлена инструкция для скриншота

2. **`createChartsSection(data)`**
   - Удалена заглушка "[ГРАФИК 1: Временной ряд]"
   - Реализована подготовка данных через `prepareChartData()`
   - Созданы таблицы:
     - `createTimeSeriesTable()` - временной ряд
     - `createWindRoseTable()` - роза ветров
     - `createTurbulenceTable()` - турбулентность

3. **`createFlightWindowsSection(data)`**
   - Использует реальные данные из `analyzed.summary.flightWindows`

4. **`createRecommendationsSection(data)`**
   - Использует `WeatherModule.generateRecommendations()`

5. **`getStyles()`**
   - Добавлен стиль `subsectionTitle`

6. **Шрифты**
   - Добавлен шрифт `monospace` для координат

### 📌 Примечание

Для добавления скриншота карты:
1. Откройте приложение с маршрутом
2. Нажмите `Win+Shift+S` (Windows) или `Cmd+Shift+4` (Mac)
3. Выделите область карты
4. Вставьте скриншот в PDF редакторе в отведённое место
