Данные по высотам (250м, 350м, 450м, 550м) рассчитываются на основе:
- `wind10m` и `wind500m` - для ветра
- `temp2m` - для температуры (с поправкой на высоту)
- `pressure` - для давления (с поправкой на высоту)
- `humidity` - для влажности (с поправкой на высоту)
