## 📊 Что реализовано

### Вертикальный профиль для БВС

**Стандартные высоты:**
- 250м
- 350м
- 450м
- 550м

**Параметры для каждой высоты:**
- 🌡️ Температура (градиент -6.5°C/км)
- 🌬️ Ветер (+18% на 500м)
- 💧 Влажность (уменьшается с высотой)
- 📊 Давление (барометрическая формула)
- 💨 Точка росы
- ❄️ Обледенение
- 🌪️ Турбулентность
- 🎯 Интегральный риск

---

## 🎯 Функции

### 1. Интерполяция по высоте

**Файл:** `js/weather.js`
**Функция:** `interpolateForAltitude(data, altitude)`

**Расчёты:**
```javascript
// Температура (градиент -6.5°C/км)
temp = temp2m + (-6.5/1000) * altitude

// Ветер (+18% на 500м)
wind = wind10m * (1 + altitude/500 * 0.18)

// Влажность (уменьшается)
humidity = humidity2m * (1 - altitude/2000)

// Давление (барометрическое)
pressure = pressure_msl * exp(-altitude/8500)
```

### 2. Вертикальный профиль

**Файл:** `js/weather.js`
**Функция:** `getVerticalProfile(analyzedData)`

**Возвращает:**
```javascript
{
    profile: [
        {altitude: 250, temp: +4.5, wind: 9.1, ...},
        {altitude: 350, temp: +3.2, wind: 10.5, ...},
        {altitude: 450, temp: +2.0, wind: 12.0, ...},
        {altitude: 550, temp: +0.8, wind: 13.5, ...}
    ],
    optimalAltitude: 250,
    dangerZone: [550],
    recommendations: [...],
    windShear: 4.4,
    tempInversion: false
}
```

## ⚠️ Ограничения

1. **Упрощённая модель** — не учитывает локальные эффекты
2. **Стандартная атмосфера** — градиент -6.5°C/км (может отличаться)
3. **Нет учёта облачности** — только cloudCover fraction
4. **Одна точка** — профиль для одной координаты
