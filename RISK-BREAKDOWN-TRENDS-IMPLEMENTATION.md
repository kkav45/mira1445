## 📊 Что реализовано

### 1. Детализация рисков по категориям ⭐⭐⭐

**Файл:** `js/weather.js`
**Функция:** `calculateRiskBreakdown()`

**Категории:**
- 🌬️ **Ветер** (0-3 балла)
- 🌧️ **Осадки** (0-3 балла)
- 👁️ **Видимость** (0-3 балла)
- ❄️ **Обледенение** (0-3 балла)
- 🌪️ **Турбулентность** (0-3 балла)

**Статусы:**
- ✅ `good` — риск отсутствует (0 баллов)
- ⚠️ `caution` — предупреждение (1 балл)
- ⚠️ `warning` — предупреждение (2 балла)
- 🔴 `critical` — критический (3 балла)

---

### 2. Тренды и тенденции ⭐⭐⭐

**Файл:** `js/weather.js`
**Функции:**
- `calculateTrends()` — расчёт трендов
- `getTrendVector()` — вектор изменения
- `getRiskChangeReason()` — причина изменения риска
- `getTrendSummary()` — сводка трендов

**Параметры:**
- **Ветер** — изменение на X% за 3 часа
- **Температура** — изменение на X% за 3 часа
- **Влажность** — изменение на X% за 3 часа
- **Видимость** — изменение на X% за 3 часа

**Векторы трендов:**
- ⬆️ `increasing` — увеличивается
- ⬇️ `decreasing` — уменьшается
- ➖ `stable` — стабильно

**Точки перелома:**
- Когда изменится риск
- Когда достигнут порог (ветер >10 м/с, обледенение high)

**Шаг 2 (Анализ):**
- [ ] Под рекомендациями видна секция "Детализация рисков"
- [ ] 5 карточек (Ветер, Осадки, Видимость, Обледенение, Турбулентность)
- [ ] Цвета соответствуют статусу (зелёный/жёлтый/красный)
- [ ] Значения отображаются (8.2 м/с, 0.0 мм, 10 км, low, low)

**Шаг 3 (Пилот):**
- [ ] После применения коррекции видна детализация
- [ ] Данные обновляются после коррекции

## 📊 Примеры использования

### Сценарий 1: Пилот видит причину риска

**До:**
```
Риск: 3.2 (medium)
```

**После:**
```
Риск: 3.2 (medium)

Детализация:
├─ Ветер: 2/3 ⚠️ (10.5 м/с)
├─ Осадки: 0/3 ✅ (0.0 мм)
├─ Видимость: 0/3 ✅ (8 км)
├─ Обледенение: 1/3 ⚠️ (medium)
└─ Турбулентность: 0/3 ✅ (low)

💡 Ветер — основная причина риска!
```

**Решение пилота:**
- "Ветер превышен, но другие параметры OK"
- "Можно лететь на меньшей высоте"
- "Нужно проверить батарею (обледенение)"

---

### Сценарий 2: Пилот видит динамику

**До:**
```
Ветер: 8.2 м/с (текущий)
```

**После:**
```
Ветер: 8.2 м/с ⬆️ +28% за 3 часа

Тренды:
⚠️ Ветер усилится на 28%
⚠️ Температура понизится на 40%

Точки перелома:
├─ 14:00 — Ветер превысит 10 м/с
└─ 18:00 — Риск обледенения high
```

**Решение пилота:**
- "Сейчас можно, но через 2 часа будет плохо"
- "Лучше полететь сейчас, а не вечером"
- "Взять тёплую одежду для электроники"

---

### Сценарий 3: Планирование времени полёта

**Точки перелома:**
```
🕐 Точки перелома:
├─ 10:00 — Старт (риск 2.1 🟢)
├─ 12:00 — Ветер усилится (риск 3.5 🟡)
├─ 14:00 — Ветер >10 м/с (риск 4.2 🟡)
└─ 16:00 — Обледенение high (риск 5.8 🔴)
```

**Оптимальное окно:**
- **Старт:** 10:00-11:00 (риск 🟢)
- **Завершение:** до 12:00 (до усиления ветра)

## ⚠️ Известные ограничения

1. **Тренды на 3 часа** — не показывают долгосрочную динамику
2. **5 категорий рисков** — могут быть перегружены для новичков
3. **Точки перелома** — показывают только ближайшие 12 часов
